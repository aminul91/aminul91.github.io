Smart Lift Managment System

The project was developed at Computer interfacing and Robotics lab of Military Institute of Science and Technology as an undergraduate project of our group and it is Smart Lift System. We used Micro C language and ATmega8 (8-bit AVR Microcontrollers) where User can provide input through Voice. Here Voice recognition techniques were implemented to analyze the input and it was converted to the binary input to provide microcontroller. Machine learning classifier K-Nearest Neighbor(KNN) was implemented to detect the actual input from various types of input patterns.As voice input may be differe for various level of sound and ML is conducive to learn various types of input and convert them as actual input.As programming language Java was used to implement ML.
